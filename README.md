# PanoramaCL

Monorepo inicial para la plataforma de descubrimiento de panoramas.

## Estructura base

- apps/web: frontend (Next.js/React)
- apps/api: backend (Node.js/NestJS)
- packages/shared-types: contratos y tipos compartidos
- packages/config: configuraciones compartidas (eslint/tsconfig, etc.)
- infra/docker: recursos de infraestructura local
- docs: arquitectura, API y lineamientos

## Estado actual

- Regla del proyecto definidas en docs y .github.
- Estructura de carpetas base creada.
- Sin codigo de aplicacion aun (siguiente paso: scaffold tecnico).
