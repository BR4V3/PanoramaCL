# apps/api

Backend de PanoramaCL.

## Estructura interna

- src/modules/locales: dominio de locales
- src/modules/panoramas: dominio de panoramas
- src/modules/search: busqueda y filtros
- src/modules/admin: operaciones administrativas
- src/common: middlewares, filtros, utilidades comunes
- src/database: ORM, migraciones y seeds
