# apps/web

Frontend de PanoramaCL.

## Estructura interna

- src/app: rutas/paginas
- src/components: UI reutilizable
- src/features: modulos de negocio por feature
- src/hooks: hooks de orquestacion de estado local
- src/services: clientes HTTP e integraciones
- src/lib: utilidades transversales
- src/styles: tokens y estilos globales
