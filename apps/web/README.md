# PanoramaCL Web

React + Vite frontend for exploring panoramas and activities in Santiago de Chile using Leaflet and OpenStreetMap.

## Requirements

- Node.js 18+
- npm 9+

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Project structure

- src/components: UI and map components
- src/data/data.json: Local dataset source
- src/App.jsx: App state and filtering logic
- src/styles.css: Minimal and responsive visual system
